# Security Policy

## Supported Versions

The latest `main` branch deploy is the only supported version.

## Reporting a Vulnerability

Email **security@flik.example.com** OR open a draft security advisory at
<https://github.com/.../security/advisories/new>.

**Response SLA**: 48h triage, 14d patch for criticals.

## Bug Bounty

None at the moment, but acknowledgement on `/security/hall-of-fame` for
accepted reports.

## Out-of-scope

- DoS / volumetric attacks
- Physical attacks
- Social engineering
- Reports requiring chained user interaction with no impact

## In-scope

- Authentication bypass, auth privilege escalation
- SQL injection, XSS, CSRF, SSRF, RCE, IDOR
- DRM bypass / paywall bypass
- PII leakage
